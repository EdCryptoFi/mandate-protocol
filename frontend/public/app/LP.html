Who Uses Mandate Protocol

There are 4 roles. Each sees a different view and has different authority:

CEO / CFO
Sets the rules nobody can break — the Institution Mandate. Requires two signatures. The absolute ceiling for the entire institution.
Compliance Officer
Sets day-to-day risk parameters within the Institution Mandate. Can only tighten, never loosen. Signs the Operational Mandate alongside the Treasury Manager.
Treasury Manager
Issues Agent Sessions — time-limited tokens the AI operates within. Monitors operations, responds to escalations, can pause the agent at any time.
Regulator
Read-only access to all AgentAction contracts across institutions. Sees every AI decision and its full reasoning — but not the institution's pool balances or positions.

Setting Up Your Institution

Step 1
CEO + CFO create the Institution Mandate

This is your institution's constitutional ceiling. It takes two signatures and defines:

  • Which assets the agent can ever touch (Bond, Treasury, Cash, etc.)
  • Which counterparties the agent can interact with
  • The absolute ceiling for any single operation (e.g. $500,000)
  • The absolute ceiling for total daily volume (e.g. $2,000,000)
  • The mandate expiry date

Once signed, nobody can exceed these limits without revoking and reissuing the mandate.

Step 2
Compliance + Treasury issue Operational Parameters

Sets day-to-day operating parameters. Can only be tighter than the Institution Mandate — never looser:

  • Margin call response limit (e.g. $300,000)
  • Target allocation per asset class (Bond 38%, Treasury 30%, Cash 10%)
  • Rebalance threshold — how far an asset can drift before the agent acts (e.g. 5%)
Step 3
Treasury Manager issues an Agent Session

When the Treasury Manager is ready to activate the AI agent, they issue a Session — a time-limited token:

  • Session duration (e.g. 8 hours for a trading day)
  • Which actions are permitted: CanRebalance, CanRespondToMarginCall, CanPostCollateral

The session expires automatically. It must be renewed for each operating period — this is a deliberate safety design.

What the Agent Does Automatically

Portfolio Drift → Rebalance

If Bond drifts from 38% to 42%, the agent detects the deviation, reasons about the rebalance, checks all limits, executes, and stores the AI rationale on-chain.

Incoming Margin Call → Automatic Response

Bank B sends a margin call for $180K Cash. Agent checks: within margin call limit ($300K) ✓, within daily volume ✓, asset allowed ✓. Responds immediately. Audit log created.

Action Exceeds Limits → Escalation

If a margin call arrives for $420K (above $300K limit), the agent creates an ApprovalRequest on-chain. The Treasury Manager reviews and approves or rejects.
